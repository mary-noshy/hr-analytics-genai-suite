# 📊 Enterprise HR Analytics & GenAI Chatbot Suite

An end-to-end operational talent engine combining predictive machine learning analytics, workforce cost-exposure tracking, and generative advisory tools powered by **Google Gemini 2.5 Flash**.

## 🚀 System Architecture & Features
* **Executive Dashboard (EDA):** Dynamic high-fidelity workforce insights built using Plotly.
* **Predictive Pipeline:** Class-balanced Stratified Random Forest pipeline filtering out uninformative identifiers and highlighting global attrition drivers.
* **Financial Loss Modeler:** Computes company-wide turnover overhead based on model risk parameters.
* **GenAI Strategic Advisor:** Conversational agent built with the `google-genai` SDK featuring strict safety guardrails and one-click copy-pasteable policy document creators.
